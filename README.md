# Ife Bakare Portfolio Website

Personal portfolio website for business analytics, product, AI, and healthcare operations projects.

## Sections
- About
- Featured Projects
- Skills
- Experience Highlights
- Contact

## How to use
1. Replace LinkedIn and GitHub links in `index.html`.
2. Add your resume as `Ife_Bakare_Resume.pdf` if you want a resume button.
3. Upload the folder to a GitHub repository.
4. Turn on GitHub Pages from repository settings.
